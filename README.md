# SearchSage: Your Smart AI Agent for Efficient Web Searching

Welcome to **SearchSage**, an AI-powered browsing agent designed to revolutionize how we search and retrieve information from the web. 🚀

![SearchSage Banner](./assets/searchsage_banner.png)

---

## 🔍 What is SearchSage?
SearchSage is an intelligent AI agent that takes your queries and scours the web to provide **clear, concise, and accurate answers**. Forget manually digging through endless search results—SearchSage does it all for you, quickly and efficiently.

### Key Features:
1. **Natural Query Processing:** Simply type your query in natural language, and SearchSage will handle the rest.
2. **Web Browsing via Tavily:** Uses Tavily’s advanced search capabilities to provide accurate and updated information.
3. **Real-Time Search:** Get answers instantly, powered by fast and intelligent processing.
4. **Clear & Concise Responses:** No fluff, just the most relevant and actionable results.
5. **Scalable & Containerized:** Fully containerized with **Docker** for easy deployment and scaling.
6. **Extendable:** Built on the **LangChain ecosystem** and **LangGraph** to integrate additional tools as needed.

---

## 🛠️ Technologies Behind SearchSage
Here’s what makes SearchSage powerful and reliable:

- **LangGraph:** Manages workflows and optimizes the processing pipeline for efficient data retrieval.
- **LangChain Ecosystem:** Handles query understanding, memory, and tool management, making the agent smarter with every interaction.
- **FastAPI:** The backbone of SearchSage’s backend, ensuring high-performance API responses.
- **Docker:** Containerizes the entire application for seamless deployment.
- **Tavily:** Enables precise and efficient web searches.

## 🚀 How to Get Started
Want to try out SearchSage? Here’s how:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Abdullah-1121/search-Sage.git
   ```

2. **Build the Docker Image:**
   ```bash
   docker build -t searcsage .
   ```

3. **Run the Backend:**
   ```bash
   docker run -p 8000:8000 searchsage
   ```


5. **Access the App:** Open your browser and go to `http://127.0.0.1:8501` to interact with SearchSage.
